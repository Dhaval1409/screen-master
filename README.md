# 🎬 ScreenMaster

ScreenMaster is an AI-powered screenplay generation platform that helps creators, writers, and filmmakers create industry-level screenplays from simple ideas, topics, or prompts.

Users can generate complete screenplays by providing:
- A simple idea
- Topic or story concept
- Desired screenplay length

It also allows users to upload their existing scripts, and the AI improves, restructures, and converts them into professional screenplay format.

---

# 🚀 Features

- ✨ AI-generated industry-standard screenplays
- 📝 Create screenplay from topic or idea
- 📚 Generate screenplay based on desired length
- 📤 Upload existing scripts for enhancement
- 🎭 Professional screenplay formatting
- 🤖 AI-powered story enhancement and restructuring
- ⚡ Fast and modern UI
