# 🎬 ScreenMaster

ScreenMaster is an AI-powered screenplay generation platform designed for creators, writers, and filmmakers to create industry-standard screenplays from simple ideas, prompts, or story concepts.

Users can generate complete screenplays by providing:

* A simple idea
* Story topic or concept
* Desired screenplay length

The platform also allows users to upload existing scripts, which are then enhanced, restructured, and converted into professional screenplay format using AI.

---

# 🚀 Features

* ✨ AI-generated industry-standard screenplays
* 📝 Create screenplays from ideas or prompts
* 📚 Generate scripts based on desired length
* 📤 Upload existing scripts for enhancement
* 🎭 Professional screenplay formatting
* 🤖 AI-powered story restructuring and improvements
* ⚡ Fast, modern, and responsive UI
